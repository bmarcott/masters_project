Fitting a digit model to an input image.

Project Abstract

Automated handwriting detection remains an interesting yet challenging problem in the Vision field. This project investigates an approach from Hinton et. al. that uses an elastic model to recognize digits. Each digit class is represented by a cubic spline in an "ideal" configuration. To classify a test image, an iterative algorithm performs an elastic match between the test image and each digit model. Validation is performed against the publically-available handwritten digit dataset, MNIST.
